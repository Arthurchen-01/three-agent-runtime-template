# Agent 1 Bootstrap

Role: architect.

## Copy

Copy `openclaw.json` to your OpenClaw config location.

## Fill in

- model id
- Feishu app id
- Feishu app secret

## Clone target

Clone this repository into:

- `C:/Users/<you>/.openclaw/workspace-agent1`

## Start

Run:

- `powershell -ExecutionPolicy Bypass -File bootstrap/agent1/launch.ps1`
