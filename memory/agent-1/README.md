# Agent 1 Memory

Agent 1 is the command center.

Memory layers:

- `short-term/` for immediate intake and dispatch notes
- `mid-term/` for current milestone and rolling planning summary
- `long-term/` for stable architecture and orchestration lessons

Legacy folders:

- `daily/` can be treated as old short-term notes
- `summaries/` can be treated as old mid-term notes
